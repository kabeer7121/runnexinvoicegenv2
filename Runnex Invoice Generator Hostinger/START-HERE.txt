After you extract this ZIP into your website root (usually public_html, or your
subdomain's root folder), do these 3 steps:

1) DATABASE
   - In hPanel: create a MySQL database and user.
   - Open phpMyAdmin → select that database → Import → choose the file:
     database-import.sql

2) API CONFIG
   - In the "api" folder, COPY "secrets.sample.php" and RENAME the copy to:
     secrets.php
   - Edit secrets.php and enter your MySQL host, database name, user, password.
   - If your site uses HTTPS, keep cookie_secure as 1.
   - For PDF rate-con parsing, add your Anthropic API key (optional).

3) PERMISSIONS
   - Make sure the "uploads" folder exists and is writable (755 or 775).

Default admin login (change password in the app after first login):
  admin / admin123
  Add agents from Admin → Agent Management (no seeded agent account).

Open in the browser: /login (or /login.html — redirects to /login)

If /api/... returns 404, see .htaccess comments (RewriteBase for subfolders).
